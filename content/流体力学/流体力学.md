# 1 第1章流体性质与分类（Properties of Fluids）

## 1.1 绪论（introduction）

### 1.1.1 流体力学（<span style="color:blue">What is fluid mechanics</span>）
流体力学所涉及的“流体”指的是液体，其具有自由变形的特征。流体的运动状态被称为“流动”，而流体力学就是一门对流体的力学性综合运动进行解析的学科。

我们生活在空气中，也知道在水深较深的地方会有不同的体验，地球上存在着水和空气等流体，正是这些流体使得生命得以诞生以及生物能够进化。

从人类诞生之初，流体就与人类紧密相连。世界四大文明都在河流附近发展，这并非偶然。早期人类过着迁徙、狩猎和采集的生活，之后逐渐过渡到以农耕为基础的定居生活。河流孕育出肥沃优质的土壤，引入灌溉技术后，农业生产得以增加，人口也随之增长。当城镇发展起来后，灌溉和防洪工程变得复杂，污水处理也成为必要，人们还开凿运河，通过船只进行金属、木材和货物的交易，这推动了文明的发展。可以说，河流的存在以及流体技术是古代文明得以发展的重要因素。此后，在以农耕为基础的文明中，流体技术以积累经验智慧的形式被继承下来。18世纪的产业革命，以机械工业为中心推动了文明的进一步发展。此后，利用流体产生能量，并将能量转化为电能等，流体在其中发挥着重要作用。蒸汽机就是将热能转化为蒸汽这种流体的能量，进而再转化为机械能的装置。如今，商用电力大多由流体能量产生，如水力发电、火力发电、核能发电、风力发电等 。在图1.6中展示了将流体能量转换为机械能（发电机轴转动的能量）的过程。

上下水道、城市燃气、石油输送、隧道通风、工厂的通风等，这些使用流体进行输送的领域都是流体力学的应用实例。除此之外，流体力学在科学技术领域广泛应用。例如，作为机械要素基础的润滑技术，以及油压设备、气压设备等在机械中的应用；在交通运输机械方面，像火箭（图1.3）、飞机、船舶、铁道、单轨电车（图1.4）、汽车等，随着速度的提高，减轻空气阻力也成为了重要课题。

空气动力噪声和振动的防止是流体力学中的重要问题。在建筑和土木工程领域，随着设施的大型化和高度化，防风对策以及空调设备等问题与流体力学相关。此外，在化学工程、医工学、生命工学、生物学、电子与电工学、体育工程学等众多领域，流体都与日常生活和工业生产密切相关。流体力学的应用范围广泛，是机械工程学中的主要科目。在21世纪，由于环境和能源问题，循环型社会的形成成为重要课题，水和大气的循环是其中的重要要素，充分利用具有其他功能的流体至关重要，因此流体力学比以往更加重要。

## 1.2 流体的基本性质（<span style="color:blue">properties of fluids</span>）

### 1.2.1 密度与比重量（<span style="color:blue">density and specific weight</span>）
密度（<ruby>密度<rt>みつど</rt></ruby>）指的是物质单位体积的质量。若某种物质的体积为$V$，质量为$M$，那么该物质的密度$\rho$可表示为：

$$\rho=\frac{M}{V}=\frac{质量}{体积} \tag{1.1}$$ 

这里，密度的单位是$[kg/m^{3}]$，密度是状态量，其值由物质的种类、温度和压力决定。
重量$G = Mg$（其中$M$为质量，$g$为重力加速度），质量$M$、体积$V$、密度$\rho = M/V$，比重量$\gamma = G/V=\rho g$。

密度的倒数称为比体积（<ruby>比体積<rt>ひたいせき</rt></ruby>），用$v$表示，其表达式为：

$$v=\frac{1}{\rho} \tag{1.2}$$ 

另一方面，比重量（<ruby>比重<rt>ひじゅう</rt></ruby>）是指物质单位体积的重量。这里的重量（重力）是指重力的大小，质量为$M$的物体，其重力$G = Mg$。对于某种物质，若其比重量为$\gamma$，重量为$G$，体积为$V$，则有：

$$\gamma=\frac{G}{V}=\frac{重量}{体积} \tag{1.3}$$ 

由$G = Mg$的关系可得：

$$\gamma=\rho g \tag{1.4}$$ 

通常，比重量在工程单位制（参照1 - 4 - 1项）中使用，单位为$[kgf/m^{3}]$。
质量为$1g$，密度$\rho = 1g/cm^{3}$，边长为$1cm$的正方体，其质量换算后$\rho = 1000kg/m^{3}$，边长变为$10cm$（即$1$分米），质量为$1000kg$，边长为$1m$。

这里我们来考虑水的密度和比重量。在$1$个大气压、$4℃$的条件下，水的密度$\rho = 1000 kg/m^{3}$，实际上密度会随温度和压力的变化而变化，但通常在一定范围内可近似认为是恒定的，因此常使用$\rho = 1000 kg/m^{3}$这个值。将其乘以重力加速度$9.81m/s^{2}$，可得水的比重量$\gamma = 1000 kgf/m^{3}$（工程单位制）$ = 9810 N/m^{2}$（国际单位制SI）。表1.1列出了一些流体的密度。

| 流体种类 | 密度$(kg/m^{3})$ |
| --- | --- |
| 水$(20°C)$ | 998.2 |
| 乙醇 | 789 |
| 海水 | 1010 - 1050 |
| 水银$(20°C)$ | 13546 |
| 石油（灯用） | 800 - 830 |
| 干燥空气$(20°C)$ | 1.205 |
| 二氧化碳$(0°C)$ | 1.977 |

在日常生活中，我们可能不太会意识到空气的密度，但实际上空气也是有质量的，是具有密度的。空气的密度会随压力、温度和湿度的变化而变化。在$1$个大气压、$20℃$的干燥空气条件下，其密度$\rho = 1.205 kg/m^{3}$，那么$1m^{3}$空气的质量约为$1.2kg$。正因如此，氢气球能够在空气中漂浮，这是因为空气的密度大于氢气的密度。


### 1.2.2 粘度与运动粘度（<span style="color:blue">viscosity and kinematic viscosity</span>）
流体具有一种特殊性质，即 “粘性（<ruby>粘性<rt>ねんせい</rt></ruby>）”。粘性是指使流体发生变形时，与变形速度相关的一种性质。例如，将浓稠的糖浆和水混合时，相比水，搅拌糖浆需要更大的力，这就表明糖浆的粘性更强。气体（如空气）也具有粘性，空气阻力就是由粘性引起的。

| 流体种类 | 粘度$(Pa·s)$ |
| --- | --- |
| 水 | $100.2×10^{-5}$ |
| 乙醇 | $119.7×10^{-5}$ |
| 水银 | $156×10^{-5}$ |
| 干燥空气 | $1.82×10^{-5}$ |
| 二氧化碳 | $1.47×10^{-5}$ |
| 流体种类 | 运动粘度$(m^{2}/s)$ |
| --- | --- |
| 水$(0°C)$ | $1.792×10^{-6}$ |
| 水$(20°C)$ | $1.004×10^{-6}$ |
| 干燥空气$(0°C)$ | $13.22×10^{-6}$ |
| 干燥空气$(20°C)$ | $15.01×10^{-6}$ |

粘度（<ruby>粘度<rt>ねんど</rt></ruby>）是表示粘性强弱的物性值，也被称为粘性系数（<ruby>粘性係数<rt>ねんせいけいすう</rt></ruby>）。

假设有两块间隔为$H$的平行平板，中间充满流体。如图1.10所示，使其中一块平板以速度$U$平行移动，此时，为了使平板移动所需的力会因流体的种类和平板的速度而变化。粘性越强的流体，所需的力越大。因此，可以根据这个力来定义粘度。当平板的速度较小时，流体的速度分布呈直线状，这种流动被称为库埃特流（<span style="color:blue">Couette flow</span>）。若平板所受的力为$F$，平板的面积为$A$，则剪切应力（单位面积上的剪切力$=\frac{F}{A}$）与$\frac{U}{H}$成正比，这一关系已通过实验得到确认，即：

$$\tau=\mu \frac{U}{H} \tag{1.5}$$
这里，比例常数$\mu$就是粘度，它是由流体的种类、温度和压力决定的物性值，单位为$[Pa·s]$。

一般情况下，物体表面附近的流体速度分布并非总是直线状，如图1.11所示，其速度分布可能呈曲线状。此时，流体所受的剪切应力$\tau$可表示为：

$$\tau=\mu \frac{du}{dy} \tag{1.6}$$ 
其中$u$是流体的流速，$y$是垂直于流动方向的坐标，$\frac{du}{dy}$被称为速度梯度（<ruby>速度勾配<rt>そくどこうばい</rt></ruby>），它是与流体变形速度相关的值。式（1.6）表明，剪切应力$\tau$与变形速度成正比，这一关系被称为牛顿粘性定律（<span style="color:blue">Newton's law of friction</span>）。

粘度$\mu$与密度$\rho$的比值称为运动粘度（<ruby>動粘度<rt>どうねんど</rt></ruby>）$v$，单位为$[m^{2}/s]$，即：

$$v=\frac{\mu}{\rho} \tag{1.7}$$
在考虑流动现象中粘性的影响时，常使用运动粘度$v$，它是粘度$\mu$（与粘性相关）与密度$\rho$（与质量相关）的比值。表1.2和表1.3列出了一些典型流体的粘度和运动粘度值。

**例题1.1**

两块平板之间充满水，保持间隔为$0.50mm$。如图1.10所示，上平板为边长$100mm$的正方形，以$20mm/s$的恒定速度移动，假设平板间的流动为库埃特流，温度为$20℃$，求上平板受到水的抵抗力$F$是多少？

**解答**

由表1.2可知，$20℃$时水的粘度$\mu = 1.0×10^{-3} Pa·s$。根据式（1.5），剪切应力为：

$$\tau = 1.0×10^{-3}(Pa·s)×\frac{20×10^{-3}(m/s)}{0.5×10^{-3}(m)} = 4.0×10^{-2}(Pa)$$
将剪切应力乘以面积即可得到力$F$：

$$F = 4.0×10^{-2}(Pa)×(0.100(m))^{2}= 4.0×10^{-4}(N)$$
### 1.2.3 体积弹性系数与压缩率（<span style="color:blue">bulk modulus of elasticity and compressibility</span>）
对气体加压时，气体会体积收缩；而对于液体和固体，在一定程度的压力下也会发生体积变化。这种因压力变化而导致体积变化的性质被称为压缩性（<ruby>圧縮性<rt>あっしゅくせい</rt></ruby>）。

如图1.12所示，某种物质在压力$p$下体积为$V$，当压力增加$dp$变为$p + dp$时，体积变为$V + dV$（其中$dV < 0$）。当压力变化很微小时，压力变化$dp$与体积变化$dV$有如下关系：

$$dp=-K \frac{dV}{V} \tag{1.8}$$
其中，$\frac{dV}{V}$被称为体积应变（<ruby>体積ひずみ<rt>たいせきひずみ</rt></ruby>），表示体积变化的比例。$K$被称为体积弹性系数（<ruby>体積弾性係数<rt>たいせきだんせいけいすう</rt></ruby>），单位为$[Pa]$。体积弹性系数$K$的倒数$\beta$称为压缩率（<ruby>圧縮率<rt>あっしゅくりつ</rt></ruby>），即：

$$\beta=\frac{1}{K} \tag{1.9}$$
体积弹性系数$K$越大，表示使物体发生相同体积变化所需的压力变化越大，即物体越不容易发生体积变化；反之，压缩率$\beta$越大，表示物体越容易发生体积变化。一般来说，液体的体积弹性系数$K$比气体大，压缩率$\beta$比气体小。
### 1.2.4 表面张力（<span style="color:blue">surface tension</span>）
液体分子间的凝聚力使它们相互吸引，当液体与其他物体接触时，其界面总是存在一种收缩的力。就如同气球总是保持一定形状一样，这种存在于流体界面的力被称为表面张力（<ruby>表面張力<rt>ひょうめんちょうりょく</rt></ruby>），单位为$[N/m]$。在失重状态的宇宙飞船中，漂浮在空中的水呈球形，落在地面上的水滴呈半球状凸起，这都是表面张力作用的结果。
表面张力的大小会因接触的流体不同而变化。例如，在$20℃$时，与空气接触的不同液体的表面张力如表1.4所示。

| 液体/接触流体 | 表面张力$(N/m)$ |
| ------- | ----------- |
| 水/空气 | 0.072 |
| 乙醇/氮气 | 0.022 |
| 水银/水 | 0.38 |

**例题1.2**
如图1.13所示，将一根细管垂直立于水面，管内水面会升高（毛细现象）。设水的密度为$\rho$，管的内径为$d$，表面张力为$T$，水面与管壁接触处的夹角（接触角，<ruby>接触角<rt>せっしゅくかく</rt></ruby>）为$\theta$，重力加速度为$g$，求水面上升的高度$h$。
**解答**
在图1.13中，高度为$h$的水柱所受重力与表面张力相平衡，即

  

$$\rho g\frac{\pi d^{2}}{4}h = \pi dT\cos\theta$$
由此可得
$$h=\frac{4T\cos\theta}{\rho gd}$$
## 1.3 流体的分类（<span style="color:blue">classification of fluids</span>）
### 1.3.1 粘性流体与非粘性流体（<span style="color:blue">viscous and inviscid fluid</span>）
流体的一个重要性质是“粘性（<ruby>粘性<rt>ねんせい</rt></ruby>）”。根据粘性的有无可以对流体进行分类，具有粘性（粘度$\mu\neq0$）的流体称为粘性流体（<ruby>粘性流体<rt>ねんせいりゅうたい</rt></ruby>），没有粘性（粘度$\mu = 0$）的流体称为非粘性流体（<ruby>非粘性流体<rt>ひねんせいりゅうたい</rt></ruby>）。
除了在极低温（2.17K以下）下液体呈现出无粘性的超流态这种特殊情况外，实际存在的流体都具有粘性，严格来说都是粘性流体。但在实际应用中，根据粘性影响的大小进行分类，当粘性的影响较大，需要考虑粘性时，将流体视为粘性流体；当粘性的影响较小，可以忽略粘性时，则将流体当作非粘性流体来处理。
例如，如图1.14所示物体周围的流动，在物体表面附近，粘性的影响较大，需要将流体当作粘性流体来处理；而在距离物体一定距离的地方，粘性的影响较小，可以将流体当作非粘性流体。因此，对于同一种流体，根据粘性影响的不同，其处理方式也会发生变化。在粘性可以忽略的情况下，数学处理会变得简单很多。
这里，用来表征流体粘性的物性值是粘度或运动粘度，而决定流动中粘性影响程度的是雷诺数（<span style="color:blue">Reynolds number</span>，<ruby>レイノルズ数<rt>れいのるずすう</rt></ruby>）$Re$，它是一个无量纲量。若流体的运动粘度为$v$，流动的特征速度（作为基准的速度）为$U$，流动区域的特征长度（作为基准的尺寸）为$L$，则雷诺数$Re$定义为：
$$Re=\frac{UL}{v} \tag{1.10}$$
雷诺数$Re$表示惯性力与粘性力的比值，该值越大，表明粘性的影响越小。
在两个几何相似的流动中，如果雷诺数相等，那么粘性力与惯性力的比例相等，这两个流动就是相似的。也就是说，这两个流动的流线相似，本质上是相同的流动，这一规律被称为雷诺相似准则（<span style="color:blue">Reynolds' law of similarity</span>）。在进行模型实验时，需要使模型和实际物体的雷诺数相等（如图1.15）。
### 1.3.2 牛顿流体与非牛顿流体（<span style="color:blue">Newtonian and non - Newtonian fluids</span>）
对于粘性流体，可以根据牛顿粘性定律（式（1.6））是否成立来进行分类。在一些流体中，剪切应力$\tau$与速度梯度$\frac{du}{dy}$成正比，这类遵循牛顿粘性定律的流体被称为牛顿流体（<ruby>ニュートン流体<rt>にゅーとんりゅうたい</rt></ruby>），空气、水以及许多油类都属于牛顿流体。而不遵循牛顿粘性定律的流体则被称为非牛顿流体（<ruby>非ニュートン流体<rt>ひにゅーとんりゅうたい</rt></ruby>）。
如图1.16所示的流动曲线，它表示了剪切应力与速度梯度之间的关系。通过原点的直线所代表的流体为牛顿流体，直线的斜率即为粘度。流动曲线不是通过原点的直线的流体，即不满足牛顿粘性定律的流体，都属于非牛顿流体。从图中可以看出，非牛顿流体有多种类型。
宾汉流体（<span style="color:blue">Bingham fluid</span>）属于塑性流体（<ruby>塑性流体<rt>そせいりゅうたい</rt></ruby>），当速度梯度为0，即变形速度极小时，仍需要一定的剪切应力才能使其发生流动，因此这种流体可以保持形状，粘土、沥青等属于这类流体。
假塑性流体（<ruby>擬塑性流体<rt>ぎそせいりゅうたい</rt></ruby>）在速度梯度为0时，剪切应力也为0 ，能够保持形状。并且当速度梯度较小时，粘度较大；随着速度梯度增大，粘度变小。高分子溶液和高分子熔体属于假塑性流体。
膨胀性流体（<ruby>ディレイタント流体<rt>でぃれいたんとりゅうたい</rt></ruby>）则相反，随着速度梯度增大，粘度增大。适量比例混合的砂和水属于这类流体。
非牛顿流体种类繁多，对于每一种非牛顿流体都需要特定的数学处理方法。但在本书中，将主要针对牛顿流体这一基本流体进行讲解。
像生面团、软膏、炼乳、鞋底橡胶等，都属于非牛顿流体，它们能够保持形状。生面团等只需用水溶解即可，血液属于假塑性流体，具有流速越快粘度越小的特性。
### 1.3.3 可压缩流体与不可压缩流体（<span style="color:blue">compressible and incompressible fluids</span>）
这里从压缩性的角度对流体进行分类。将需要考虑压缩性影响的流体称为可压缩流体（<ruby>圧縮性流体<rt>あっしゅくせいりゅうたい</rt></ruby>），而压缩性影响较小，可以忽略压缩性的流体称为不可压缩流体（<ruby>非圧縮性流体<rt>ひあっしゅくせいりゅうたい</rt></ruby>）。
表征流体压缩性的物性值是体积弹性系数$K$或压缩率$\beta$，而决定流动中压缩性影响程度的是马赫数（<span style="color:blue">Mach number</span>，<ruby>マッハ数<rt>まっはすう</rt></ruby>）$M$。马赫数$M$定义为流动的特征速度$U$与音速（<ruby>音速<rt>おんそく</rt></ruby>，<span style="color:blue">sound velocity, speed of sound</span>）$a$的比值，即：
$$M=\frac{U}{a} \tag{1.11}$$
其中，音速$a$可以由流体的体积弹性系数$K$和密度$\rho$通过下式求得：
$$a=\sqrt{\frac{K}{\rho}} \tag{1.12}$$
可以根据马赫数$M$的大小来判断压缩性的影响程度。当$M< 约0.3$时，压缩性的影响较小，通常可以将流体近似看作不可压缩流体。当$M> 约0.3$时，需要考虑压缩性，此时流体常被当作可压缩流体来处理。当速度超过音速（$M\geq1$）时，压缩性的影响会显著显现，会产生激波（<ruby>衝撃波<rt>しょうげきは</rt></ruby>，<span style="color:blue">shock wave</span>），导致剧烈的变化。当$M>1$时的流动称为超音速流动（<ruby>超音速流れ<rt>ちょうおんそくながれ</rt></ruby>，<span style="color:blue">supersonic flow</span>）（例如图1.17），当$M<1$时的流动称为亚音速流动（<ruby>亜音速流れ<rt>あおんそくながれ</rt></ruby>，<span style="color:blue">subsonic flow</span>）（例如图1.18）。
由此可见，压缩性对流体的影响并非直接由流体的体积弹性系数$K$（或压缩率$\beta$）决定，对于同一种流体，随着速度的变化，其可能被当作可压缩流体，也可能被当作不可压缩流体，处理方式也会相应改变。
**例题1.3**
在1个大气压和常温下，空气在什么流速下可以当作不可压缩流体来处理？
**解答**
在温度为$t/[^{\circ}C]$、1个大气压下，干燥空气的音速$a/[m/s]$为
$$a = 331.45 + 0.607t$$
在常温下，当马赫数$M$为0.3时，对应的速度约为100m/s。因此，当空气流速在100m/s以下时，可以忽略其压缩性，将其当作不可压缩流体来处理。

### 1.3.4 理想流体（<span style="color:blue">ideal fluid</span>）
既无粘性又无压缩性的流体被称为理想流体（<ruby>理想流体<rt>りそうりゅうたい</rt></ruby>）。理想流体由于没有粘性，不存在能量损失和阻力。然而，实际存在的流体与理想流体存在矛盾之处。
比较物体表面附近理想流体和实际流体的流动情况，如图1.19所示。（a）理想流体不会因粘性而受到剪切应力，在物体表面流体可以自由流动。而（b）实际流体由于粘性的影响，在物体表面流速为0 ，在表面附近会形成一个流速较慢的区域，这个区域被称为边界层（<ruby>境界層<rt>きょうかいそう</rt></ruby>，<span style="color:blue">boundary layer</span>，详见91节）。边界层外侧的流动称为主流（<ruby>主流<rt>しゅりゅう</rt></ruby>，<span style="color:blue">main flow</span>），在主流中粘性影响较小，可以近似看作理想流体。通常情况下，边界层较薄，在大部分流动区域都可以将流体当作理想流体来处理，这样会使解析过程变得非常容易。

## 1.4 单位与量纲（<span style="color:blue">units and dimensions</span>）

### 1.4.1 单位制（<span style="color:blue">systems of units</span>）
  
在表示物理量时需要用到单位，世界上存在多种单位制。国际单位制（<ruby>国際単位系<rt>こくさいたんいけい</rt></ruby>，<span style="color:blue">International System of Units，SI</span>）是国际通用的标准单位制，为了实现国际统一，本书原则上使用SI单位制。
SI单位制通过表1.5所示的7个基本单位的组合来表示所有物理量的单位。在流体力学中，常用的基本单位有长度$[m]$、质量$[kg]$、时间$[s]$。通过这些基本单位的组合，可以得到面积$[m^{2}]$、速度$[m/s]$、力的单位$[N]=[kg\cdot m/s^{2}]$等。

| 量 | 名称 | 符号 |
| --- | --- | --- |
| 长度 | 米 | $m$ |
| 质量 | 千克  $kg$ |
| 时间 | 秒 | $s$ |
| 电流 | 安培 | $A$ |
| 热力学温度 | 开尔文 | $K$ |
| 物质的量 | 摩尔 | $mol$ |
| 发光强度 | 坎德拉 | $cd$ |

另一方面，在产业界还存在一种以往常用的工程单位制（<ruby>工学単位系<rt>こうがくたんいけい</rt></ruby>，<span style="color:blue">gravitational units</span>，也称为重力单位系），它以长度$[m]$、力$[kgf]$、时间$[s]$为基本单位。SI单位制与工程单位制的关系为：力$1 kgf = 9.81 N$，这里的$1kgf$是质量为$1kg$的物体所受重力的大小。由于重力加速度的大小会随地点不同而变化，并且随着宇宙开发的推进，在无重力环境中，工程单位制的使用受到限制，逐渐不再常用。

在欧美，还经常使用英制单位（<ruby>英国単位系<rt>えいこくたんいけい</rt></ruby>，<span style="color:blue">British gravitational units，略记BG单位系，English units</span>），其长度单位为英尺$[ft]$，力的单位为磅力$[lbf]$。SI单位制与英制单位的关系为：长度$1 ft = 0.3048 m$（$1 ft = 12 inch$），力$1 lbf = 4.4482 N$，质量$1 lbm = 0.4536 kg$，$1 slug = 14.5939 kg$（$lbm$是质量单位磅）。温度差$1 K = 1.8^{\circ}F$，也有$1 K = 1.8^{\circ}R$（$R$为兰金温标），这些关系在书末的附录中有详细说明。

尽管存在多种单位制，但目前国际上统一采用SI单位制。

**例题1.4**

有两个外观完全相同的球，一个质量为1kg，另一个质量为10kg。在地球上，通过用手拿起它们可以很容易地区分，但在失重状态的宇宙飞船中该如何区分呢？

**解答**
在失重状态下，两个球的重量都为0kgf，无法通过重量来区分。但是，它们的质量分别为1kg和10kg，当推动这些球时，根据牛顿第二运动定律，要使它们获得相同的加速度，所需的力相差10倍，因此可以通过施加力并观察球的运动情况来区分它们。
### 1.4.2 量纲（<span style="color:blue">dimension</span>）
  
如前所述，SI单位制的基本思想是“所有物理量都可以通过7个基本单位的组合来表示”。这里，将基本单位对应的基本量长度用“$L$”、质量用“$M$”、时间用“$T$”、温度用“$\Theta$”表示，这些被称为量纲（<ruby>次元<rt>じげん</rt></ruby>）。
通过这些量纲的组合，可以表示其他物理量的量纲。例如，面积的量纲为$[L^{2}]$，速度的量纲为$[LT^{-1}]$，力是质量与加速度的乘积，所以力的量纲为“$LMT^{-2}$”。
一般来说，描述物理现象的表达式在任何单位制下都应该成立，这种与单位制无关的方程式被称为完整方程（<ruby>完全方程式<rt>かんぜんほうていしき</rt></ruby>，<span style="color:blue">complete equation</span>）。假设有$n$个物理量$A_{1}, A_{2}, \cdots, A_{n}$，它们之间的关系可以用完整方程$f(A_{1}, A_{2}, \cdots, A_{n}) = 0 \tag{1.13}$表示，如果该方程由$m$个基本单位组成，根据白金汉π定理（<ruby>バッキンガムのπ定理<rt>ばっきんがむのぱいていり</rt></ruby>，<span style="color:blue">Buckingham's π theorem</span>），式（1.13）可以转换为关于$n - m$个无量纲数$\pi_{1}, \pi_{2}, \cdots, \pi_{n - m}$的关系式$F(\pi_{1}, \pi_{2}, \cdots, \pi_{n - m}) = 0 \tag{1.14}$。这里的无量纲数$\pi_{1}, \pi_{2}, \cdots, \pi_{n - m}$可以表示为$m + 1$个以下物理量的幂次乘积，被称为π项。
